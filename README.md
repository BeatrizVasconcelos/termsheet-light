#  :orange_circle: Termsheet Light

This project consists of a web application for managing real estate deals. Hope you enjoy it! :blush:

## :desktop_computer: Running the project
- Run `npm install`
- Run `ng serve`
- Navigate to `http://localhost:4200/`

## :rocket: Technologies

- Angular 16.1.4
- NgRx
- RxJs
- Angular Material
- Jest
